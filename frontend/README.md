# Restaurant Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
REACT_APP_API_URL=http://localhost:5000/api
```

3. Start the application:
```bash
npm start
```

## Build for Production

```bash
npm run build
```

## Features

- Responsive design
- Tailwind CSS styling
- React Router for navigation
- Context API for state management
- Protected routes
- Toast notifications
